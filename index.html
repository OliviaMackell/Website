Bits Vs Bytes

A bit (lowercase 'b') is the smallest unit of digital data, representing a 0 or 1. A byte (uppercase 'B') is a group of 8 bits,

typically representing a single character. One pixel requires an 8 bit system to be able to go from 0-255,

which is why our computers run on these sets of 8 bits and it a reacurring number that when dealing with tech you'll see often.

An example of this byte system for letters and symbols is the letter "A", or:

01000001

But that A is specifically capitilzed, a lowercase "a" has a different symbol:

01100001

All following a 8 bit, or byte system.

Learning Binary

Learning binary is no simple task, but it isn't like a complacated lanague, just a combination of numbers put together!

For example, try to write "Hello" in binary, it looks like as follows:

01001000 01100101 01101100 01101100 01101111

There is a simple pattern to follow and you can either learn it from a graph like found on Ascii-Code.com:

The Magic Binary Trick

Ask these four questions in order:

Q1: Is your number 8 or higher? YES = write 1 NO = write 0

Q2: Is your number (or what's left) 4 or higher? YES = write 1 NO = write 0

Q3: Is your number (or what's left) 2 or higher? YES = write 1 NO = write 0

Q4: Is your number (or what's left) 1 or higher? YES = write 1 NO = write 0

Read your four digits top-to-bottom. That is their number in binary. Convert it and you have their number.
